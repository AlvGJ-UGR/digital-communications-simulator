# 📘 Teoría del sistema de comunicaciones digitales

## 📡 Modelo de sistema

Un sistema de comunicación digital básico se compone de:

1. Fuente de información (bits)
2. Modulador
3. Canal de transmisión
4. Demodulador
5. Receptor

---

## 📶 Canal AWGN

El canal se modela como:

r(t) = s(t) + n(t)

donde:
- s(t): señal transmitida
- n(t): ruido gaussiano blanco con media cero

---

## 📊 Relación SNR

SNR = Eb / N0

donde:
- Eb: energía por bit
- N0: densidad espectral de potencia del ruido

---

## 📉 BER

El BER mide la probabilidad de error en la transmisión.

Es la métrica principal para comparar esquemas de modulación.
